Alur Kerja & Cara Kerja Sistem

Aplikasi ini menghubungkan Kamera Webcam di Web Browser (Antarmuka) dengan Model Pengenal Isyarat Tangan berbasis Python (Sistem Pemroses) secara langsung. Berikut adalah diagram alur pemrosesan data:

1. Penangkapan Kamera

Sistem mengakses webcam menggunakan standar HTML5 getUserMedia() dan menampilkannya di halaman web.

2. Pengiriman Frame

Secara berkala, citra dari video disalin ke penampung digital, dikodekan ke dalam format Base64, dan dikirimkan via protokol HTTP.

3. Pemrosesan Citra

Sistem memotong area ROI tangan, memisahkan warna kulit tangan dari latar belakang, dan mengubah ukuran gambar menjadi 28x28 piksel grayscale.

4. Pencocokan Karakter

Model pengenal memproses gambar 28x28 untuk memprediksi kecocokan huruf (36 kelas) dan mengembalikannya ke dalam bentuk data JSON.

Catatan Keandalan: Pembatasan area deteksi pada kotak ROI tetap menjamin stabilitas pembacaan tangan dan ramah penggunaan daya komputasi dibandingkan model pelacakan seluruh frame.
